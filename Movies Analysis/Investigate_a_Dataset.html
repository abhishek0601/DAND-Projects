Introduction¶

The dataset contains observations of more than 10,000 movies collected from the Movie Database(TMDb). It has 21 columns including revenue,budget,genres,popularity.

Questions to be discussed

  • What is the no. of movies produced with respect to genres?
  • Does the buget have any relation with the profit earned?
  • Does the revenue have any relation with the profit earned?
  • Does the duration of movie have any relation with the profit earned?
  • Is there an increase in profit with the increase in years?
  • Is there a relation between popularity and profit?
  • What relation does votes have with the popularity?
  • Is there any change in the amount of movies produced with increase in year?
  • Movies count of various cast.
  • Which is the most popular movie and highest gross profited movie?

  • Correlation among various other features
  • Analysing hit movies

    In [23]:
    #Stores all the genres and gives the count of each genre
def category_count(column,tmdb_df):
    #Taking each column and separate string by '|'
    genres = tmdb_df[column].str.cat(sep = '|')
    
    #Making it a series and adding it separately
    genres = pd.Series(genres.split('|'))
    
    #Arrange the genres in descending order
    genre_count = genres.value_counts(ascending = False)
    
    return genre_count
